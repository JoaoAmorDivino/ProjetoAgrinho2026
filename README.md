# 🌱 Agro Forte, Futuro Sustentável | Agrinho 2026

> "O equilíbrio perfeito entre a produção de alimentos para o mundo e a preservação do nosso meio ambiente."

Este projeto é um site educacional desenvolvido para o concurso **Agrinho 2026**, no estado do Paraná. O objetivo principal é informar sobre as práticas sustentáveis que estão transformando o agronegócio paranaense e brasileiro, provando que é possível produzir com alta tecnologia e preservar a natureza simultaneamente.

---

## 🚀 Funcionalidades e Destaques

* **Conteúdo Educativo:** Informações diretas e fundamentadas sobre Sistema de Plantio Direto (SPD), Integração Lavoura-Pecuária-Floresta (ILPF), Controle Biológico e Gestão da Água.
* **Design Concretista:** Interface gráfica inspirada no movimento concretista. Utiliza formas geométricas, alto contraste, cores sólidas (verde, vermelho, preto e off-white) e tipografia forte, transmitindo a solidez e a precisão do Agro 4.0.
* **Navegação Fluida:** Efeito de *smooth scroll* implementado para uma experiência de usuário mais agradável ao clicar nos menus.
* **Animações de Entrada (Scroll Reveal):** Os blocos de conteúdo surgem na tela de forma interativa conforme o usuário rola a página para baixo.
* **Easter Egg Oculto:** Uma mensagem educativa especial escondida no console do navegador (pressione F12 para descobrir).

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando os fundamentos do Desenvolvimento Web (Front-end):

* **HTML5:** Estruturação semântica do conteúdo para melhor acessibilidade e leitura.
* **CSS3:** Estilização visual, uso de variáveis CSS e layout em *CSS Grid* adaptável.
* **JavaScript (Vanilla):** Interatividade de página, manipulação do DOM e uso da *Intersection Observer API* para otimizar as animações de rolagem.

---

## 💻 Como Executar o Projeto

Como o site foi criado com tecnologias web nativas, você não precisa instalar nada de complexo para visualizá-lo:

1. Faça o download da pasta do projeto.
2. Abra a pasta no seu computador.
3. Dê um clique duplo no arquivo `index.html`.
4. O site será aberto automaticamente no seu navegador padrão (Google Chrome, Edge, Firefox, etc.).

**Dica para desenvolvedores (VSCode):** Para editar o código e ver as mudanças em tempo real, abra a pasta no Visual Studio Code e utilize a extensão **Live Server**.
